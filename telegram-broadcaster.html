Telegram Broadcaster

Send messages to multiple Telegram users efficiently

Quick Guide
1. Settings: Configure your bot token (stored in browser)
2. Recipients: Add recipient IDs (comma-separated or one per line)
3. Compose: Write your message with formatting
4. Send: Use "Send Test" first, then "Start Broadcast"

Keyboard Shortcuts: Ctrl+B (Bold), Ctrl+I (Italic), Ctrl+U (Underline)
Rate Limit: ~30 messages per second with 1s batch delay
